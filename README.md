# ♻️ Deteksi Jenis Sampah dengan Deep Learning

Aplikasi web berbasis Streamlit untuk mendeteksi jenis sampah:
- 🍂 Organik
- 🗑 Anorganik
- ♻️ Daur Ulang

## 📦 Teknologi
- Streamlit
- TensorFlow (MobileNetV2)
- OpenCV, Pillow
- Python 3.9+

## 🚀 Cara Menjalankan

```bash
pip install -r requirements.txt
streamlit run app.py
